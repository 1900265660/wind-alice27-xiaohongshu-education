# AI 研报初稿核验清单

| 检查项 | 我会怎么问自己 | 为什么要查 |
|---|---|---|
| 数据来源 | 这个数字来自哪里？是公告、研报、数据库，还是新闻？ | 来源不清，后面的判断就不稳 |
| 数据时间 | 这个数据是哪一年、哪个季度的？ | 旧数据可能撑不起新结论 |
| 数据口径 | 是单季度还是全年？国内还是全球？批发还是零售？ | 口径不同，数字不能直接比 |
| 前后一致 | 同一个指标前后有没有换说法？ | 防止初稿里出现自相矛盾 |
| 估值假设 | 增长率、利润率、折现率是不是太乐观？ | 估值不是只看结果，更要看假设 |
| 证据关系 | 结论是不是从数据推出来的？ | 语言顺不等于逻辑成立 |
| 事实/预测 | 有没有把预测写成已经发生的事实？ | 金融内容要区分已发生和未发生 |
